Discord Wiki Doc

Discord Wiki can be used to publish embeds to Discord. The first message will be the index, followed by each wiki entry in a message.

User Interface

Webhook
Discord webhook URL configured in channel settings.

Wiki Name
Will be used as file name when exporting the wiki.

Index Title
Title used in the index message.

Index Link Name
If set, an embed will be added to each wiki entry linking back to the index message. When changed, all messages will be updated.

Thumbnail URL
Thumbnail used for all messages. When changed, all messages will be updated.

Server ID
First number in a message URL. Has to be set manually since the webhook doesn't return it.

Channel ID / Index Message ID
IDs of the channe and the index message. Can't be edited since the index message needs to be created by the bot belonging to the webhook.

Import / Export buttons
Used to store the wiki settings and content as a file. This is necessary when working with multiple wikis and strongly recommended when working with one. For convenience the data is also stored in the browser, so it's not always necessary to import. But the data should always be exported.

Wiki Entries
List of wiki entries and button to create a new one.

Title
Title of the selected wiki entry.

Text area below Title
Content of the selected wiki entry.

Image URL
If set, the image will be added to the selected wiki entry.

Message URL
If the message is already published, the link can be copied to the clipboard to be pasted as a link in another wiki entry.

Delete
Deletes the selected wiki entry. If it is already published, the Discord message has to be deleted manually.

Limitations

  • There is not special support for working with multiple wikis. It can be achieved by exporting the wikis to files and importing the one to be edited. To initialize a new wiki, open Discord Wiki in a private tab.
  • One Discord message can contain up to 6000 characters. The index message will reach that (due to the lengthy message links) after about 50-60 entries. Discord Wiki will show a warning and refuse to publish in that case. A new wiki has to be started, which can use the same webhook to be created below the previous wiki.
  • One Discord embed can contain up to 2000 characters, so the wiki entries are limited accordingly.
